**Repositorio estrategia GeoMIP**  

Este repositorio contiene las implementaciones originales de los métodos asociados  con el artículo  "GeoMIP: A Geometric-Topological and Dynamic Programming Framework for Enhanced Computational Tractability of Minimum Information Partition in Integrated Information Theory"

En la carpeta **/data** se encuentran los archivos correspondientes a redes de diferentes tamaños.  

Para las redes de mas de 15 nodos se usa el aplicativo creation.py  para generarlas.  

En la carpeta **/results** se encuentran los archivos de excel  con los resultados de tiempo y de valor de perdida, tanto para el Método 1(GPU Accelerated) como para el Método 2 (Dynamic Programming reformulation) de la estrategia GeoMIP.  

La carpeta  **/src** presenta la subcarpeta /Method1_GPU_Accelerated que presenta el código fuente  para esta versión de la estrategia GeoMIP y la subcarpeta /Method2_Dynamic_Programming_reformulation  con el código fuente para esta versión de la estrategia GeoMIP
