Repositorio estrategia GeoMIP
En la carpeta /data se encuentran los archivos correspondientes a redes de diferentes tamaños.
Para las redes de mas de 15 nodos se usa el aplicativo creation.py  para generarlas.
En la carpeta /resultas se encuentran los archivos de excel  con los resultados de tiempo y de valor de perdida, tanto para el Método 1(GPU Accelerated) como para el Método 2 (Dynamic Programming reformulation) de la estrategia GeoMIP.
